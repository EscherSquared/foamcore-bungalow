TCJ Foam Core Bungalow Project
Grade 9 culminating project: students build a one-storey bungalow scale model from a single sheet of foam core. Includes three designs (two easy, one medium), printable cut lists, instruction sheet, reflection page, and assessment rubric.
What's in this repo
```
website/
├── index.html              ← Main page: 3 bungalow designs with drawings & cut lists
├── documents.html          ← Download page for the 4 student/teacher PDFs
├── styles.css              ← Stylesheet (blueprint aesthetic)
└── pdfs/
    ├── TCJ_Bungalow_Instructions.pdf   (2 pages)
    ├── TCJ_Bungalow_Cutlists.pdf       (3 pages — one design per page)
    ├── TCJ_Bungalow_Reflection.pdf     (2 pages)
    └── TCJ_Bungalow_Rubric.pdf         (1 page)
```
